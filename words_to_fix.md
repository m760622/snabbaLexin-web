## كلمات مشكوك فيها تحتاج استبدال:

### كلمات غير صحيحة أو غير شائعة:
- TAM → استبدالها بكلمة أخرى
- SOT → استبدالها بكلمة أخرى  
- ORK → استبدالها (ليست كلمة سويدية شائعة)
- SUB → استبدالها (ليست سويدية)
- MEH → استبدالها (ليست سويدية)
- SUH → استبدالها (ليست سويدية)
- KOB → استبدالها (ليست شائعة)
- ARF → استبدالها (ليست سويدية)
- VOR → استبدالها (ليست شائعة)
- GÅT → مشكوك فيها
- TÅB → ليست شائعة
- KAFÉ → يجب أن تكون CAFÉ بالإنجليزية أو نستخدم كلمة أخرى
